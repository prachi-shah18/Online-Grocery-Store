# Online Grocery Store

## Project Overview
The Online Grocery Store is a C++ based project developed using the Singly Linked List data structure.  
This system allows users to manage grocery items dynamically by adding, deleting, and viewing items in a shopping cart.

---

## Features
- Add grocery items
- Remove grocery items
- Display cart items
- Dynamic cart management
- Checkout functionality

---

## Technologies Used
- C
- Data Structures
- Singly Linked List

---

## Project Objective
The main objective of this project is to demonstrate the practical implementation of linked list operations in a real-world shopping cart system.
